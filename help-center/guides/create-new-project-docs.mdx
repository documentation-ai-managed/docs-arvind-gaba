## Overview

Set up your first docs project on Arvind Gaba.

## Steps
<Steps>
  <Step title="Go to Dashboard">
    Log in and open `https://arvindgaba.com/dashboard`.
  </Step>
  <Step title="Create New Project">
    Click "New Project", enter a name, and select a template.
  </Step>
  <Step title="Organize Pages">
    Add, edit, or delete pages as needed in your new project.
  </Step>
  <Step title="Publish">
    Click "Publish" to make docs live for your team.
  </Step>
</Steps>
