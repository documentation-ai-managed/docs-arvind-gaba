## Overview

To begin, create a documentation project from your dashboard.

## Steps

1. Sign in at `https://arvindgaba.com/dashboard`
2. Click "New Project"
3. Name your project and select a template
4. Click "Create"

Your project will appear in the Docs list on the dashboard.

## Next

Start adding pages using the editor provided.
