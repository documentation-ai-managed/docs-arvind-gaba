## Prerequisites

Before you begin, ensure you have the following:

<Callout kind="info">

- A modern web browser like Chrome, Firefox, or Safari
- An email address for account verification
- Optional: GitHub account for version control integration

</Callout>

## Create Your Account

Sign up for Arvind Gaba in under a minute. Follow these steps to register and log in.

<Steps>
  <Step title="Visit the Dashboard" icon="monitor">
    Navigate to `https://dashboard.example.com` in your browser.
  </Step>
  <Step title="Register" icon="user-plus">
    Click **Sign Up**, enter your email and password, then submit the form.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox for a verification link from Arvind Gaba and click it.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to the dashboard, enter your credentials, and access your account.
  </Step>
</Steps>

## Create a New Documentation Space

Organize your projects by creating dedicated documentation spaces.

<Steps>
  <Step title="Select New Space" icon="plus">
    From the dashboard, click **New Documentation Space**.
  </Step>
  <Step title="Name Your Space" icon="edit-3">
    Enter a name like `My Project Docs` and add a brief description.
  </Step>
  <Step title="Choose Template" icon="layout">
    Select **Blank Space** or **Markdown Starter** for your initial setup.
  </Step>
  <Step title="Create and Enter" icon="check-circle">
    Click **Create**. You now land in your new space's editor.
  </Step>
</Steps>

## Upload Your Initial Content

Add your first documents using the built-in tools. Choose your preferred method.

<Tabs>
  <Tab title="File Upload" icon="upload">
    Drag and drop Markdown files or use the upload button.
    
    Supported formats: `.md`, `.mdx`, `.txt`, images (`.png`, `.jpg`).
    
    <Image
      src="https://via.placeholder.com/600x300/3B82F6/white?text=Upload+Files"
      alt="File upload interface showing drag-and-drop area"
      width="600"
      height="300"
    />
  </Tab>
  <Tab title="Markdown Editor" icon="edit">
    Use the live editor to write directly.
    
````markdown
# Welcome to My Project

## Overview

This space organizes all project documentation.

- **Installation**: Follow the quickstart guide
- **API**: See endpoints at `https://api.example.com`
````
  </Tab>
  <Tab title="API Upload" icon="code">
    Use the REST API for programmatic uploads.
    
    <CodeGroup tabs="cURL,JavaScript">
````bash
curl -X POST https://api.example.com/v1/spaces/{SPACE_ID}/docs \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "First Doc",
    "content": "# Hello World\nContent here."
  }'
````
````javascript
const response = await fetch('https://api.example.com/v1/spaces/{SPACE_ID}/docs', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    title: 'First Doc',
    content: '# Hello World\nContent here.'
  })
});
````
    </CodeGroup>
  </Tab>
</Tabs>

<Callout kind="tip">
  Replace `{SPACE_ID}` with your actual space identifier from the dashboard URL and `YOUR_API_KEY` with your generated key.
</Callout>
